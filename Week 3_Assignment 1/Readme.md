# Netflix User Analytics

This project analyzes Netflix user behavior using Machine Learning techniques. The dataset contains user demographics, viewing habits, subscription details, and spending patterns. The project includes Exploratory Data Analysis (EDA), data preprocessing, and the implementation of Decision Tree, K-Nearest Neighbors (KNN), and Linear Regression models.

The primary objectives are to predict subscription renewal, estimate monthly spending, and identify key factors that influence customer retention. The insights generated from this analysis can help Netflix improve user engagement, reduce churn, and make data-driven business decisions.
